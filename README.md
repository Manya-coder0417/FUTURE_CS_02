# FUTURE_CS_02 – Phishing Email Detection & Awareness System

## Internship Track
Cyber Security – Future Interns

## Task Objective
The objective of this task is to analyze phishing email samples, identify common phishing indicators, classify emails based on risk, and create user awareness to help prevent phishing attacks.

This task focuses on understanding social engineering techniques used in phishing emails and explaining them in simple, business-friendly language.

---

## Scope of Work
- Analysis of simulated phishing email samples
- Identification of phishing indicators
- Email classification (Safe / Suspicious / Phishing)
- User awareness and prevention guidelines
- No interaction with malicious links or attachments

---

## Methodology
Publicly available and simulated phishing email samples were reviewed manually. Each email was analyzed without clicking on links or downloading attachments. The analysis focused on sender details, message content, urgency indicators, and suspicious URLs.
